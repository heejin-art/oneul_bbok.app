## 2. [개발지니] 효율 전략 — 1 게임 = 1 코드베이스 = 웹+앱

### 핵심 결정
**같은 코드로 웹과 앱을 동시에 출시한다.** 별도 네이티브 앱 코드 작성 안 함.

```
[게임 1개]
   ↓
React + Vite + TypeScript + Three.js (PWA)
   ↓
   ├── 웹 배포 → Vercel/Netlify (도메인 = 웹사이트)
   └── PWA → PWABuilder.com / Bubblewrap CLI → APK → Google Play
```

### 기술 스택 (양 게임 공통)
- **빌드:** Vite (속도)
- **언어:** TypeScript (실수 방지)
- **3D:** Three.js (vanilla — R3F는 번들 크기 부담)
- **PWA:** vite-plugin-pwa (manifest + service worker)
- **사운드/햅틱:** Web Audio API + navigator.vibrate
- **호스팅:** Vercel (무료 + CI 자동)
- **APK 패키징:** PWABuilder.com → 사이닝키만 준비

### 공유 자산
- `shared/design-tokens/` — 컬러·타이포·이징·모션 변수
- `shared/3d-utils/` — Three.js 헬퍼(조명·카메라·셰이더 베이스)
- `shared/pwa-config/` — PWA 설정 보일러플레이트

## 3. [디자이니] 비주얼 컨셉 (1 디자인 시스템, 2 테마)

### 공통 디자인 시스템
- **무드:** 2026 K-감성 — 글래스모피즘 + 소프트 그라데이션 + 파스텔 네온
- **타이포:** Pretendard Variable (한글) + Inter Variable (영문)
- **모션 이징:** Cubic Bezier `(0.32, 0.72, 0, 1)` — *부드러운 새티스파잉*
- **사운드:** ASMR 톤 — 저음역 부드러운 팝, 햅틱 동기화

### 🅳 버블팝랩 테마
- **컬러:** Aqua/민트 + 화이트 글래스 + 분홍 액센트
- **3D 무드:** 비누방울 같은 굴절·하이라이트, 파스텔 무지개
- **모션:** *부풀기 → 뽀삭 → 파편 소산*

### 🅵 스트레스 캡슐 테마
- **컬러:** 라벤더/퍼플 + 골드 액센트 + 다크 배경
- **3D 무드:** 캡슐 메탈릭 + 내부 빛, 파편이 별가루처럼 흩어짐
- **모션:** *길게 누름 → 압력 게이지 진동 → 폭발 → 메시지 페이드인*
