# PRD 초안 — 1주 게임 + 포트폴리오 웹사이트

> 작성: [기획지니] · 결재 대기: [희지니]
> 목표: 1주 내 출시 가능한 초간단 중독성 게임 1개 + 게임을 자랑할 포트폴리오 웹사이트(3D 히어로) 1개

## 0. 제약 조건 (Constraints)

- **개발 기간:** 1주
- **플랫폼:** 모바일 앱(구글 플레이) + 반응형 웹
- **체험 강도:** *머리 안 쓰는, 편한* — 한 손, 짧은 세션
- **중독성:** 계속하게 되는 루프 필수
- **심사 리스크:** 최소 — 권한 미사용, 민감 콘텐츠 없음, 광고도 처음엔 미적용 권장(심사 단순화)

## 1. 2026 트렌드 인사이트 ([기획지니] 리서치 요약)

### 한국 Z/MZ 라이프스타일
- **메타센싱** — 내면 감정·기분에 집중하는 흐름
- **제철코어** — "지금 아니면 못 누리는 희소한 경험"
- **필코노미(Feel-conomy)** — 결정은 줄이고 만족은 빠르게
- **나노 커뮤니티** — 작지만 취향 명확한 집단
- **3대 키워드:** 초몰입 · 초효율 · 초개인

### 모바일 게임 시장
- 캐주얼·퍼즐 강세 지속 (Block Blast 3억 다운로드가 증거)
- **하이브리드 캐주얼** 부상: 쉬운 코어 + 깊은 보상
- **아이들/클리커** 다운로드·매출 동반 성장
- 짧은 세션 + 럭(랜덤 보상) 요소 + AI 개인화

### 향후 2–3년 전망
- 감정·웰니스 결합 게임 (메타센싱과 결합)
- ASMR/촉감 중심 게임 (필코노미)
- 일일 한정 콘텐츠 (제철코어)
- K-감성 비주얼 글로벌 수출(K-Z 트렌드 일본·서구권 확산)

## 2. 게임 기획안 3개

### 🅰 안. **무드캐치 (MoodCatch)** — 감정 줍기
**한 줄 컨셉:** 떨어지는 컬러 감정 구슬을 손가락으로 슥슥 쓸어 같은 색 바스켓에 담는 ASMR형 캐주얼.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 구슬 낙하 → 스와이프 → 매칭 콤보 → 일일 무드 리포트 자동 생성 |
| 머리 안 쓰는 정도 | ★★★★★ (색깔 매칭만) |
| 중독 포인트 | 콤보 사운드 + 일일 감정 그래프 누적 + 친구에게 "오늘의 무드" 공유 |
| 3D/비주얼 | 글래스모피즘 구슬, 트레일 파티클 |
| 트렌드 적중 | 메타센싱 ◎ / 필코노미 ◎ / 나노커뮤니티 ◎ |
| 심사 리스크 | 매우 낮음 |
| 1주 구현 가능성 | ◎ (물리 단순) |
| 차별점 | 게임 + 감정 일기 하이브리드 — 시장에 잘 없음 |

### 🅱 안. **한 줄 정리 (Tidy Line)** — 정돈 ASMR
**한 줄 컨셉:** 흐트러진 도형 더미를 한 손가락 드래그 한 번에 한 줄로 정렬하는 미니멀 정리 게임.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 도형 배치 확인 → 드래그 → 일렬 정렬 모션 → 다음 레벨 |
| 머리 안 쓰는 정도 | ★★★★★ |
| 중독 포인트 | 정렬 사운드/햅틱 ASMR, 끝없는 레벨, 짧은 세션 |
| 3D/비주얼 | 도형이 3D로 회전·정렬되는 모션 핵심 |
| 트렌드 적중 | 초효율 ◎ / 미니멀 ◎ / 필코노미 ◎ |
| 심사 리스크 | 매우 낮음 |
| 1주 구현 가능성 | ◎ |
| 차별점 | 시장의 *정리 ASMR* 영상 인기에 비해 모바일 게임화는 적음 |

### 🅲 안. **오늘의 운 (Fortune Pop)** — 운세 톡톡
**한 줄 컨셉:** 풍선/캡슐을 톡 터트리면 안에서 오늘의 한 줄·감성 메시지·랜덤 보상이 튀어나오는 일일 의식형 게임.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 일일 풍선 N개 무료 → 탭 → 메시지/스킨 수집 → 컬렉션 |
| 머리 안 쓰는 정도 | ★★★★★ (탭만) |
| 중독 포인트 | "오늘만" 제철 한정, 컬렉션 욕구, 친구 공유 컷 |
| 3D/비주얼 | 풍선이 진짜 3D로 부풀고 터지는 물리감 |
| 트렌드 적중 | 제철코어 ◎ / 메타센싱 ○ / 나노커뮤니티 ◎ |
| 심사 리스크 | 낮음 (단, 문구를 *운세/사주*가 아닌 "데일리 메시지"로 포지셔닝) |
| 1주 구현 가능성 | ◎ (가장 단순) |
| 차별점 | 일일 의식 + 컬렉션 결합, SNS 공유 친화 |
