## 1. 2026 트렌드 인사이트 ([기획지니] 리서치 요약)

### 한국 Z/MZ 라이프스타일
- **메타센싱** 내면 감정·기분 집중
- **제철코어** "지금 아니면 못 누리는 희소한 경험"
- **필코노미** 결정은 줄이고 만족은 빠르게
- **나노 커뮤니티** 작지만 취향 명확한 집단
- **3대 키워드:** 초몰입 · 초효율 · 초개인

### 모바일 게임 시장
- 캐주얼·퍼즐 강세 지속 (Block Blast 3억 다운로드)
- **하이브리드 캐주얼** 쉬운 코어 + 깊은 보상
- **아이들/클리커** 다운로드·매출 동반 성장
- 짧은 세션 + 럭(랜덤 보상) + AI 개인화

### 스트레스 해소·피젯 카테고리 (추가 리서치)
- Pop It, 버블랩, 슬라임, 비누 컷팅, 스퀴시 등 단순 *촉감* 콘텐츠가 글로벌·국내 모두 강세
- 한국 인기 앱: 마이 오아시스, POP IT 스트레스 해소, 부드러운 스트레스 해소 ASMR
- 핵심 디자인 기둥: **즉각 만족 · 인지 부하 최소 · 스트레스 해소**
- 모드 분기: 타이머·골 없는 *명상 모드* + 도전 모드(레코드 저장)
- 학술 근거: 캐주얼 게임이 불안·우울 증상 완화에 효과 (Cyberpsychology, Behavior, and Social Networking 연구)

### 향후 2–3년 전망
- 감정·웰니스 결합 게임 (메타센싱 + 캐주얼)
- ASMR/촉감 중심 (필코노미)
- 일일 한정 콘텐츠 (제철코어)
- K-감성 비주얼 글로벌 수출

## 2. 게임 기획안 6안

> 안 표기: 🅰 🅱 🅲 = 1차 안, 🅳 🅴 🅵 = 누르기/스트레스 해소 카테고리 추가 안

### 🅰 무드캐치 (MoodCatch)
**한 줄:** 떨어지는 컬러 감정 구슬을 손가락으로 같은 색 바스켓에 담는 ASMR 캐주얼.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 구슬 낙하 → 스와이프 매칭 → 콤보 → 일일 무드 리포트 자동 생성 |
| 머리 안 쓰는 정도 | ★★★★★ |
| 중독 포인트 | 콤보 사운드 + 일일 감정 그래프 누적 + "오늘의 무드" 공유 |
| 3D/비주얼 | 글래스모피즘 구슬, 트레일 파티클 |
| 트렌드 적중 | 메타센싱 ◎ / 필코노미 ◎ / 나노커뮤니티 ◎ |
| 심사 리스크 | 매우 낮음 |
| 1주 구현 | ◎ |
| 차별점 | 게임 + 감정 일기 하이브리드 |

### 🅱 한 줄 정리 (Tidy Line)
**한 줄:** 흐트러진 도형 더미를 한 손가락 드래그 한 번에 한 줄로 정렬하는 미니멀 ASMR.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 배치 확인 → 드래그 → 일렬 정렬 모션 → 다음 레벨 |
| 머리 안 쓰는 정도 | ★★★★★ |
| 중독 포인트 | 정렬 사운드/햅틱 ASMR, 끝없는 레벨 |
| 3D/비주얼 | 도형이 3D로 회전·정렬되는 모션 핵심 |
| 트렌드 적중 | 초효율 ◎ / 미니멀 ◎ / 필코노미 ◎ |
| 심사 리스크 | 매우 낮음 |
| 1주 구현 | ○ (정렬 알고리즘 디버깅 변수) |
| 차별점 | 정리 ASMR의 게임화 |

### 🅲 오늘의 운 (Fortune Pop)
**한 줄:** 풍선/캡슐을 톡 터트리면 안에서 오늘의 한 줄·랜덤 보상이 튀어나오는 일일 의식형 게임.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 일일 풍선 N개 무료 → 탭 → 메시지/스킨 수집 → 컬렉션 |
| 머리 안 쓰는 정도 | ★★★★★ |
| 중독 포인트 | "오늘만" 제철 한정, 컬렉션 욕구, SNS 공유 컷 |
| 3D/비주얼 | 풍선이 3D로 부풀고 터지는 물리감 |
| 트렌드 적중 | 제철코어 ◎ / 메타센싱 ○ / 나노커뮤니티 ◎ |
| 심사 리스크 | 낮음 (포지셔닝을 *데일리 메시지*로) |
| 1주 구현 | ◎ |
| 차별점 | 일일 의식 + 컬렉션 결합 |

### 🅳 버블팝랩 (BubblePopLab) — *NEW*
**한 줄:** 화면 전체가 무한 뽁뽁이. 손가락으로 슥슥 누르면 3D 버블이 *뽀삭* 터지는 무한 새티스파잉 모드.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 패턴 선택 → 무한 팝 → 1분 챌린지로 베스트 기록 → 새 패턴 언락 |
| 머리 안 쓰는 정도 | ★★★★★ |
| 중독 포인트 | 뽁 사운드 + 햅틱 + 일일 패턴(제철) + 베스트 기록 갱신 |
| 3D/비주얼 | 진짜 *부풀고 터지는* 3D 버블 (셰이더/메탈볼) |
| 트렌드 적중 | 필코노미 ◎ / 메타센싱 ○ / 제철코어 ○ |
| 심사 리스크 | 매우 낮음 |
| 1주 구현 | ◎ (가장 단순) |
| 차별점 | 한국 시장의 *Pop It 류*에 3D 비주얼·일일 한정 패턴으로 차별 |

### 🅴 스퀴시팩토리 (SquishFactory) — *NEW*
**한 줄:** 떡·마시멜로·슬라임·거품 등 다양한 오브젝트를 누르면 3D로 변형되고 떼면 복원되는 촉감 컬렉션.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 오브젝트 선택 → 자유 스퀴시 → 누적 카운트로 새 오브젝트 언락 |
| 머리 안 쓰는 정도 | ★★★★★ |
| 중독 포인트 | 누적 압력 그래프, 새 재질·소리 언락, 일일 한정 재료 |
| 3D/비주얼 | 소프트바디 시뮬레이션(WebGL/Unity) — 디자이니 핵심 무대 |
| 트렌드 적중 | 필코노미 ◎ / 메타센싱 ◎ / 초개인 ◎ |
| 심사 리스크 | 매우 낮음 |
| 1주 구현 | △ (소프트바디 1개 잘 만든 후 재질 swap이면 ◎, 신규 구현이면 ○) |
| 차별점 | 진짜 *반응하는* 3D 촉감 — 2D 위주 시장 대비 비주얼 압도 |

### 🅵 스트레스 캡슐 (Stress Capsule) — *NEW*
**한 줄:** 매일 N개의 3D 캡슐 등장. 길게 누르면 캡슐이 부풀고 → 빵! 안에서 오늘의 한 줄/감성 컬러/미니 보상이 튀어나옴.

| 항목 | 내용 |
|---|---|
| 코어 루프 | 일일 캡슐 → 길게 누르기 → 압력 게이지 → 폭발 → 보상·메시지 수집 |
| 머리 안 쓰는 정도 | ★★★★★ |
| 중독 포인트 | 압력 빌드업의 카타르시스 + 일일 한정 + 컬렉션 |
| 3D/비주얼 | 캡슐 부풀기 셰이더 + 폭발 파편 |
| 트렌드 적중 | 제철코어 ◎ / 메타센싱 ◎ / 필코노미 ◎ |
| 심사 리스크 | 낮음 |
| 1주 구현 | ◎ |
| 차별점 | 🅲 오늘의 운 + 🅳 팝의 결합 — 빌드업 → 폭발의 의식감 |

## 4. 팀 권고 ([각 페르소나 입장])

### [기획지니]
1순위 **🅵 스트레스 캡슐** — 누르기 카타르시스 + 일일 의식 + 컬렉션, 메타센싱/제철코어/필코노미 트리플 적중.
2순위 **🅴 스퀴시팩토리** — 차별성·비주얼 최강, 단 소프트바디 시뮬 변수.
3순위 **🅳 버블팝랩** — 가장 안전한 출시 카드.

### [개발지니]
- 1주 안 *안정 출시* 최우선이면 **🅳 > 🅵 > 🅰 > 🅲 > 🅴 > 🅱**.
- 🅴는 소프트바디 라이브러리(예: matter.js + soft-body 플러그인 or three.js + ammo.js)에 익숙해질 시간 1.5일 필요. 1개 마스터하고 재질 swap 전략이면 가능.
- 🅵는 *길게 누르기 게이지 + 폭발 파편 셰이더*로 비주얼 카타르시스를 가장 효율적으로 뽑음.

### [디자이니]
- **🅴 스퀴시팩토리** — 2026 K-감성 비주얼(글래스·소프트바디·파스텔) 진가 발휘, 히어로 3D 섹션이 *그대로* 게임 비주얼이 됨.
- **🅵 스트레스 캡슐** — 빌드업 → 폭발의 모션 시퀀스가 히어로 섹션에 임팩트 ◎.
- 🅳도 좋지만, 비주얼 *기억에 남는 정도*는 🅴·🅵가 앞섬.

## 5. 통합 권고

> **🅵 스트레스 캡슐**을 1차 추천, **🅴 스퀴시팩토리**를 차순위로.
>
> 두 안 모두 *누르면 스트레스 풀리는* 사용자 의도와 정확히 일치. 🅵는 일일 의식 + 컬렉션 메타로 *재방문 동력*까지 확보. 🅴는 비주얼 임팩트 최강이지만 1주 일정에 소프트바디가 변수.
>
> 안전한 출시 우선이면 **🅳 버블팝랩**, 임팩트 최대화면 **🅴**, 균형이면 **🅵**.
