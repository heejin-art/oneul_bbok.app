# WOOHEEJIN

우희진(희지니) 팀의 가상 협업 저장소.
1주 안에 *초간단·중독성* 게임 2종을 웹+앱으로 출시한다.

## 산출물

| 게임 | 컨셉 | 디렉터리 |
|---|---|---|
| 🅳 [버블팝랩](./bubble-pop-lab/) | 무한 뽁뽁이 + 일일 패턴 + 1분 챌린지 | `bubble-pop-lab/` |
| 🅵 [스트레스 캡슐](./stress-capsule/) | 일일 캡슐 + 길게 누르기 + 메시지 컬렉션 | `stress-capsule/` |

각 게임 = 1 코드베이스 = **웹사이트 + 모바일 앱** 동시 출시 (PWA 전략).

git clone https://github.com/kkm15921598/WOOHEEJIN.git
